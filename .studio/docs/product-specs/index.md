# Product Specifications Index

This template is a development foundation, not a product with user-facing features. The specifications below describe the developer experience capabilities provided by the template.

## Development Environment

- [Type Safety System](type-safety-system.md) — Comprehensive type checking from database to UI with zero `any` types or type assertions
- [Quality Gates](quality-gates.md) — Pre-commit hooks, ESLint rules, and test coverage requirements that enforce standards
- [Testing Infrastructure](testing-infrastructure.md) — Vitest configuration with 100% coverage requirement and integration test patterns

## API Layer

- [tRPC Integration](trpc-integration.md) — End-to-end type-safe API calls with automatic serialization and React Query integration
- [Health Check Endpoint](health-check-endpoint.md) — Basic health monitoring endpoint for deployment verification

## Frontend Foundation

- [Next.js App Router Setup](nextjs-app-router.md) — File-based routing with React Server Components and client-side navigation
- [Styling System](styling-system.md) — Tailwind CSS v4 with PostCSS integration and custom font configuration
- [Client State Management](client-state-management.md) — tRPC provider wrapping React Query for server state synchronization
