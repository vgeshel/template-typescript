# Product Specifications Index

This template provides infrastructure rather than user-facing features. Specifications document the capabilities provided to developers.

## Core Capabilities

- [Type-Safe API Layer](type-safe-api.md) — tRPC integration providing end-to-end type safety between frontend and backend
- [Quality Enforcement](quality-enforcement.md) — Pre-commit hooks, ESLint rules, and test coverage requirements that prevent defects
- [Development Workflow](development-workflow.md) — Hot reloading, test watch mode, and structured logging for rapid iteration
- [Testing Infrastructure](testing-infrastructure.md) — Vitest configuration with 100% coverage thresholds and component testing support

## Template Features

- [Project Structure](project-structure.md) — Directory organization with clear separation between app routes, server code, and shared utilities
- [Configuration Management](configuration-management.md) — Environment variable handling with dotenvx for development and test isolation
