# Glossary

**App Router** — Next.js 13+ routing paradigm using the `app/` directory with file-based routing, server components, and layouts.

**Bun** — JavaScript runtime and package manager used instead of Node.js for faster performance and built-in TypeScript support.

**httpBatchLink** — tRPC link that batches multiple procedure calls into a single HTTP request for efficiency.

**Pino** — Fast, structured JSON logger used for application logging instead of console methods.

**Pre-commit hooks** — Git hooks enforced by Husky that run typecheck, lint, and tests before allowing commits.

**publicProcedure** — tRPC procedure builder for endpoints that don't require authentication.

**SuperJSON** — Data serialization library that preserves JavaScript types (Date, Map, Set, etc.) across client-server boundaries in tRPC.

**TDD (Test-Driven Development)** — Development workflow requiring tests to be written before implementation code for all changes.

**tRPC** — TypeScript RPC framework providing end-to-end type safety between Next.js frontend and backend without code generation.

**Vitest** — Fast unit test framework compatible with Vite, used for running tests with 100% coverage requirements.

**Zod** — TypeScript-first schema validation library used for runtime validation of external data and API contracts.
